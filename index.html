YouTube Info Copy Extension

This is a simple Chrome Extension that lets you copy YouTube video details (title, channel, views, duration, comments) with one click!

How to Use:

  1. Download the ZIP file below
  2. Unzip it
  3. Go to chrome://extensions
  4. Enable Developer mode (top-right)
  5. Click Load unpacked and select the unzipped folder
  6. Start using it on YouTube!
